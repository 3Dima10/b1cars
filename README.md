# AUTOUA — сайт объявлений об авто (Django)

Готовый проект: публичная часть (каталог + карточка авто) и закрытая админ-панель
для владельца сайта (создание / изменение / удаление объявлений, смена аватара).

## Структура
- `config/` — настройки проекта Django
- `cars/` — основное приложение (модели, вьюхи, шаблоны, формы)
- `static/` — CSS/JS
- `media/` — загруженные фото авто и аватар (создаётся автоматически)

## Установка

```bash
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt

python manage.py migrate
python manage.py createsuperuser   # создать логин/пароль для входа в админку
python manage.py runserver
```

Сайт откроется на http://127.0.0.1:8000/

Вход в закрытую админ-панель: http://127.0.0.1:8000/admin-login/
(войти может только пользователь с `is_staff=True`, т.е. созданный через
`createsuperuser`, или отмеченный галочкой "staff" в стандартной
Django-админке на `/django-admin/`).

Демо-аккаунт, который уже создан в этой сборке для проверки:
- логин: `admin`
- пароль: `admin12345`

**Обязательно смените пароль / создайте своего суперпользователя перед
реальным использованием.**

## Что реализовано

- Публичный каталог машин (сетка карточек), адаптивный (3 / 2 / 1 колонки).
- Страница объявления: галерея фото, характеристики, кнопка звонка
  (`tel:` — на телефоне сразу открывает набор номера) и кнопка Telegram
  (открывает `https://t.me/<ник>` в новой вкладке).
- Бургер-меню (три полоски) справа сверху с выезжающей панелью, в которой
  находится кнопка "Admin" (ведёт на страницу входа) — вход и сама
  админ-панель доступны только зарегистрированному администратору.
- Закрытая админ-панель (`/panel/`) с аватаром (клик — смена аватара) и тремя
  кнопками: **Создать** (зелёная), **Изменить** (жёлтая), **Удалить** (красная).
  На десктопе форма/список появляются прямо под кнопками, на телефоне —
  открываются как полноэкранное модальное окно с крестиком закрытия
  (в точности как на присланных макетах).
- Загрузка нескольких фото на одно объявление.
- Подвал сайта: название, телефон, Telegram.
- Плавные анимации: появление страниц, карточек, наведение на кнопки/карточки,
  выезд бокового меню и модальных окон.

## Настройка контактов и названия сайта

Название сайта, номер телефона и Telegram-аккаунт хранятся в модели
`SiteSettings` и меняются через стандартную Django-админку
(`/django-admin/` → Настройки сайта) — там же можно поменять аватар,
если не пользоваться кликом по аватарке в `/panel/`.

## Внимание — только для локальной разработки

`DEBUG = True` и `SECRET_KEY` в `config/settings.py` подходят только для
разработки. Перед выкладкой в интернет обязательно:
- выключить `DEBUG`,
- указать реальный `ALLOWED_HOSTS`,
- задать свой `SECRET_KEY` через переменную окружения,
- настроить раздачу `media/` и `static/` через сервер (nginx и т.п.).
